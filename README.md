    + ?seq()：学习seq()的用法

    + 二目运算符也是函数
    + R向量用“*”乘，获得新向量，而不是向量内积或外积
    + 多用()避免优先级错误
    + 用seq()生成等差数列（向量）
    + 用rep()生成循环数列（向量）
    + 用any(条件)和all(条件)对向量中的元素进行判断

3. 用R处理游程问题
    + 编程中如何减少内存消耗：避免头指针反复摆动
    + 累积和函数cumsum()

    + NA为缺失值：NA无法参与运算

5. R的筛选机制（filtering）
    + 用[条件]进行筛选

    + 用which(条件)进行筛选

    + 这个方法速度较快，因为进行的是纯向量化操作
